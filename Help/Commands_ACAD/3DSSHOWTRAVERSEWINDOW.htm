3DSSHOWTRAVERSEWINDOW (Command)

Add Row
Adds a new row to the traverse table with the default values of 0° bearing and 0.000 distance.

Remove Row
Removes the currently selected row from the traverse table. Any information in the row will be lost. This process cannot be undone.

Clear Table
Completely clears the traverse table. Any information in currently in the table will be lost. This process cannot be undone.

Open Traverse
Opens a previously saved traverse from a CSV file.

Save Traverse
Saves the current traverse to a CSV file.

Set Base point
Sets the base point for the traverse.

Draw Traverse
Draws the traverse in the AutoCAD window using temporary graphics.

Select Object
Use this command to select lines in the AutoCAD window which will populate the traverse table.
Note: Only connecting lines are valid.

Zoom Extents
Zooms the AutoCAD window to the extents of the current traverse.

Feet to Meters
Converts the currently selected rows distance to meters.

Links to Meters
Converts the currently selected rows distance to meters.

Help
Displays the help file.
