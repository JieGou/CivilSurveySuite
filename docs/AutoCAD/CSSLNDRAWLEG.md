# CSSLNDRAWLEG

## Description

Draws return leg on a given line.

## Usage

* Run command (CSSLNDRAWLEG)
* Pick first point (generally picked on a line)
* Pick second point (generally picked on a line)
* Enter a distance
* Accept results, cancel or flip the point (rotate 180Â°)
