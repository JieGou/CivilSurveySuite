# CSSPTPRODDIST

## Description

Creates a point (DBPoint) at the production of a line.

## Usage

* Run Command (CSSPTPRODDIST)
* Select a line
* Enter a distance or pick on screen
* Accept results, cancel or flip the point (rotate 180Â°)
