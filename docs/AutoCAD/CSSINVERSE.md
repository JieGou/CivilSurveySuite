# CSSINVERSE

## Description

Displays detailed information between two points.

## Usage

* Run command (CSSINVERSE)
* Pick first point
* Pick second point

## Example Output

```
Command: CSSINVERSE
Pick first point:
Pick second point:
Angle: 88°30'02" (268°30'02")
Distance: 56.2792320620831
dX:-56.26 dY:-1.4725 dZ:0
Slope:0%
```
