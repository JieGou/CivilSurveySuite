# CSSLABELMASKON

## Description

Turns the label mask on for a selected label on a **CogoPoint**.

## Usage

* Run command (CSSLABELMASKON)
* Select a **CogoPoint**

## Example Output

```
Command: CSSLABELMASKON
Select CogoPoints to turn label mask(s) on: 1 found
Select CogoPoints to turn label mask(s) on:
```
